Controles:
L -> liga / desliga a luz
F -> liga / desliga o fog
W -> incrementa pitch do vagalume lider
S -> decrementa pitch do vagalume lider
D -> incrementa yaw do vagalume lider
A -> decrementa yaw do vagalume lider
Arrow Up -> incrementa velocidade do vagalume lider
Arrow Down -> decrementa velocidade do vagalume lider

O vagalume lider possui a luz azul enquanto os outros possuem uma luz verde.
Cada vagalume possui um periodo para bater asas, e enquanto elas estao se movendo para baixo,
o vagalume ganha uma forca positiva no eixo Z e, quando estao se movendo para cima, ganha uma forca negativa no eixo Z,
simulando o movimento do vagalume de acordo com as asas.
